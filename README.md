# HTML Simple App

Estrutura básica para sites estáticos. 

## Ferramentas/módulos pré-instalados

* Grunt
* Sass
* Compass
* Bower
* Foundation
* FastClick

## Instalação

### Clonar o respositório

```
$ git clone https://github.com/thiagosf/HTMLSimpleApp.git site
```

### Instalar módulos do grunt e componentes do bower

```
$ npm install
$ bower install
```
